# News Classification ML Project

## Classifying News Articles into 4 Categories Using Machine Learning

### Results
- Best Accuracy: 90.41% (Logistic Regression)
- Algorithms Compared: Logistic Regression, SVM, KNN
- Dataset: AG News (127,600+ articles)
- Categories: World, Sports, Business, Science/Tech
